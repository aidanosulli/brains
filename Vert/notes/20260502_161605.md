# Off-Season Strength & Power Session (Plyo Box Protocol)

## Core Principles

- During periods of high running load, reduce landing stress by incorporating plyo boxes while maintaining explosive training quality
- Cluster sets allow higher-intensity loading with managed fatigue (brief intra-set rest rather than reducing weight)
- Superset pairings combine stability/activation work with reactive/power output movements
- Progressive volume reduction in the squat wave (5–4–3–2) paired with consistent jump volumes potentiates power expression

## Key Frameworks & Progressions

**Upper Body Block**
- Lock Lat Pull — 2 × 12 (activation/prep)
- Bench Press from Pins (Cluster Sets) — 9 × 3, structured as 3 clusters of 3 reps with 20s rest between mini-sets
- Incline DB Prone Row — 3 × failure (performed during bench rest periods as filler)

**Lower Body Block — Superset 1 (Activation + Reactivity)**
- Knee ISO Push — 3 sec hold × 4 reps
- Reactive Drop Jumps with Single-Leg Catch — 30 sec
- Repeat × 2

**Lower Body Block — Superset 2 (Strength + Power)**
- Single Leg RDL — 2 × 8
- Seated Single Leg Broad Jump to Box — 2 × 4

**Lower Body Block — Wave Loading + Power Cluster**
- Safety Bar Box Squat with Chains — wave: 5 / 4 / 3 / 2 reps
- Seated Broad Jump to Box — 4 × 3 reps (paired with squat sets)
- BW Countermovement Jump — 4 × 1 rep (paired with each squat set)

**Accessory Finisher**
- Hip Flexor Cable Pull-In (Two-Leg) — 2 × 20

## Actionable Advice

- Use plyo boxes for jump landings when managing cumulative running load — absorbs impact while preserving power training stimulus
- Pair isometric holds (knee ISO push) directly before reactive jumps to prime the stretch-shortening cycle
- Use rest periods during heavy compound lifts productively by inserting accessory/filler work (e.g., rows during bench rest)
- Apply cluster set structure (20s intra-set rest) on pin press to sustain bar speed and intent across all reps
- Wave load the squat (descending reps with chains) alongside jumps to potentiate explosive output

## Things to Avoid

- Don't ignore accumulated running load when programming plyometrics — unmanaged landing stress increases injury risk during off-season blocks
- Avoid high-impact landings (hard floor drop jumps) when weekly running volume is already significant

## Key Takeaways

- **Manage total stress intelligently** — box landings are a simple, effective tool to preserve explosive training quality under high running loads
- **Cluster sets** on pin press allow maximal intent across all reps without technical breakdown from fatigue
- **Activation before power** — isometric holds paired with reactive jumps is an efficient priming strategy within supersets
- **Filler work maximises session density** — inserting rows during bench rest periods adds volume with no extra time cost
- The session follows a logical sequencing: activation → strength-power supersets → wave loading → accessory, which is a replicable off-season template
