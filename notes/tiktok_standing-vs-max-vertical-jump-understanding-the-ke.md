# Standing vs Max Vertical Jump — Understanding the Key Distinctions

> **Source:** https://www.tiktok.com/@jumpsbychi/video/7564848380271283487
> **Date:** 2026-06-01

CONFIDENCE: medium

## Core Principles

- "Vertical jump" is not a single metric — it simply means jumping for height (vertically)
- Different types of vertical jumps measure similar but distinct athletic abilities
- Neither type is more "legitimate" than the other; they are complementary metrics

## Key Frameworks & Progressions

**Types of Vertical Jumps:**
- **Standing vertical jump** — measured from a stationary position, no approach
- **Max vertical / approach vertical jump** — uses a running approach to generate maximum height
- **Hands-on-hips jump** — removes arm swing as a variable
- **Seated vertical jump** — isolates leg power, removes lower-body momentum

**How Major Leagues Measure:**
- **NFL** — measures standing vertical jump only
- **NBA** — measures *both* standing vertical and max vertical jump

## Things to Avoid

- Assuming a running approach jump is "cheating" or illegitimate
- Treating "vertical jump" as a single, universal measurement
- Dismissing one type of vertical jump as inferior to another

## Key Takeaways

- "Vertical jump" is a category, not a single test — always clarify *which type* is being referenced
- Standing and max vertical jumps are two different metrics measuring a related but distinct expression of the same athletic quality
- Context matters: know which standard is being applied (NFL vs NBA vs other)
- Rejecting approach jumps as invalid is like rejecting sourdough as "not real bread" — categorically flawed reasoning
